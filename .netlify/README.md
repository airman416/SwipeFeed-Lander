# Netlify Configuration

This directory is automatically created by Netlify CLI when you run `netlify init`.

## Quick Deploy Commands

```bash
# Install Netlify CLI (if not already installed)
npm install -g netlify-cli

# Login to Netlify
netlify login

# Build and deploy
npm run build:netlify
netlify deploy --prod
```

## Configuration

All Netlify settings are configured in `netlify.toml` at the root of the project.

See `DEPLOYMENT.md` for detailed deployment instructions.
